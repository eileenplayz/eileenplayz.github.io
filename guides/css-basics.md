<h1>CSS Basics</h1>
<p>CSS is used to style the HTML elements. Here's a basic example of CSS:</p>

<pre><code>h1 {
  color: blue;
  text-align: center;
}</code></pre>

<p>In the above example, we are styling the <code>&lt;h1&gt;</code> element to be blue and center-aligned.</p>
